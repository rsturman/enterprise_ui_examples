Enterprise UI Examples

A collection of enterprise-grade UI demos showcasing different design systems and application patterns.

Lovely HR Workforce Management

A full-featured workforce management dashboard with a dark theme and teal accents. Includes employee tables with search, sort, filter, and pagination; a multi-step employee creation wizard; project management with grid/list views; an interactive calendar with event scheduling; and Excel data export.

Custom Dark Theme
Vortex Energy Intranet

A corporate intranet portal built with the Atlassian Design System. Features a sticky top navigation bar, a sidebar with favorites and spaces, quick-access tool links with notification badges, and a personalized greeting dashboard for a global energy company.

Atlassian Design
Meridian & Co. Corporate Intranet

An enterprise corporate intranet using the IBM Carbon Design System with IBM Plex typography. Showcases a hero landing section, a CEO letter with editorial styling, mission and vision cards, company values in a four-column grid, and a strategic plan layout with image cards and gradient overlays.

IBM Carbon Design
NexusOne Enterprise Workforce Management

A dark-themed workforce management platform using DM Sans typography and teal accents. Features employee tables with search, sort, and pagination; shift scheduling; analytics dashboards; and Excel data export via SheetJS.

Custom Dark Theme
Lovely HR – Employees

An employee directory and management interface built with the SAP Fiori design language. Includes a master-detail layout, employee profile cards, role and status badges, and support for both light and dark themes.

SAP Fiori Design
